# browser_press_key

Presses a keyboard key or key combination.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| key | string | Yes | Key to press (e.g., "Enter", "Escape", "Tab", "Control+c") |

## Return Shape

```json
{
  "content": [{ "type": "text", "text": "Pressed key Enter" }]
}
```
